# 🛋️ Canto da Casa – E-commerce de Móveis

**Canto da Casa** é um projeto de e-commerce de móveis desenvolvido do zero, desde o protótipo até a implementação do código. O objetivo principal foi criar um site responsivo, moderno e funcional, focando na experiência do usuário e na apresentação dos produtos de forma clara e atrativa.

## 🛠️ Tecnologias Utilizadas

- **Figma** – Criação do layout e protótipo;
- **HTML5** – Estruturação do conteúdo;
- **CSS3 / SCSS** – Estilização com pré-processamento para melhor organização e manutenção;
- **Bootstrap** – Framework para responsividade e grid;
- **Slick Carousel** – Biblioteca para criação de sliders e carrosséis dinâmicos;
- **JavaScript** – Adição de interações e dinamicidade no site.

## 🎯 Funcionalidades

- Layout totalmente responsivo;
- Seções com sliders/carrosséis para produtos em destaque;
- Menu de navegação adaptável para desktop e mobile;
- Interações com JavaScript para melhorar a experiência do usuário;
- Design pensado para ser intuitivo e fácil de navegar.

## 📑 Protótipo

O layout foi desenhado no **Figma**, onde foram definidos os componentes, paleta de cores, tipografia e estrutura de seções.
